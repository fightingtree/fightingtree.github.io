linux基本指令

系统操作

reboot 重启

文件操作

cd 文件夹名 //进入文件夹内
mkdir 文件夹名 //创建一个文件夹(文件夹名在终端标识为蓝色)
touch 文件名 //创建一个文件(文件夹名在终端标识为白色)
rm -f 文件名 //删除一个文件 
rmdir 文件夹名 //删除一个文件夹
mv 文件名1 文件夹名2 //将文件1移到文件夹2中
gcc -c test.c //编译一个叫test.c的c语言文件，最终会生成一个test.o文件
gcc -o test test.o //生成一个名叫test的可执行文件
g++ test.cpp -o test//编译一个叫test.cpp的cpp文件并生成可执行文件test
./test //执行可执行文件


chmod + xx.py  //更改py文件的执行权限
./xx.py  //执行py文件

vim

Linux操作系统下的文本编辑器

打开vim

打开linux终端，输入指令

vim 文件名  //若文件名对应文件存在则打开对应文件，若无对应文件则新建一个文件

vim三模式

  • 命令行模式：输入vim后最初进入的模式，可以进行粘贴复制但不能进行编辑，按ESC键后进入命令状态，可在此状态下输入命令对vim文件进行操作
  • 插入模式：命令行模式按下按下i(insert)后进入的模式，可以进行编辑，按ESC键后返回命令行
  • 底行模式：插入模式按ESC键进入命令行模式后输入保存命令

vim指令

vi保存命令

   :q 退出没有修改的文件
   :q! 强制退出，且不保存修改过的部分
   :w 保存文件但不退出（相当于windows的ctrl S）
   :x 保存文件并退出
   :w 名字 将文件重命名保存
   :wq

vi撤销

在命令行模式下输入u

vim设置

中文乱码问题

  1. 编辑.vimrc文件
    vim .vimrc
  2. 在.vimrc中加入以下文件
    set fileencodings=utf-8,ucs-bom,gb18030,gbk,gb2312,cp936
set termencoding=utf-8
set encoding=utf-8
  3. 执行生效
    source .vimrc
    以上参考博客链接

####　viｍ代码高亮
在.vimrc文件中加入

syntax on
